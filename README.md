# Offline Dialogue Labeler Pro — Python 3.7.7 compatible build

Эта версия проекта подготовлена специально под сборку на **Windows + Python 3.7.7**.

Что уже учтено:
- совместимость кода с Python 3.7;
- сборка через **PyInstaller 5.13.2**, совместимый с Python 3.7;
- `src` подключается как python-пакет через `--paths src`, а не как data-папка;
- есть проверка `tkinter` перед сборкой;
- если в корпоративном индексе нет пакетов, можно положить рядом папку `wheelhouse/` и собрать офлайн.

## Что умеет приложение

- читает входные файлы из `input/`:
  - `.xlsx`
  - `.xlsm`
  - `.csv` UTF-8 с разделителем `;`
- использует только колонки `session_id` и `text`;
- разбивает `text` на реплики по разделителю `//`;
- поддерживает несколько режимов разметки;
- есть встроенный конструктор режимов;
- в конструкторе можно дублировать поля и правила;
- одно правило может сразу управлять несколькими полями в блоке `then`, а `require_empty` реально блокирует их в UI;
- автосохранение каждые `N` секунд;
- фильтр «только неразмеченные»;
- быстрый переход к записи по `session_id`;
- экран статистики;
- история значений и автодополнение;
- обязательные поля и простые межполевые правила;
- экспорт результата в Excel.

## Структура

- `run_app.py` — точка входа
- `src/labeler/` — код приложения
- `config/modes.json` — конфиг режимов
- `config/app_settings.json` — настройки приложения
- `input/` — входные файлы
- `output/` — результат и служебные данные

## Локальный запуск из Python

```bat
py -3.7 -m venv .venv
.venv\Scripts\activate.bat
python -m pip install -r requirements.txt
python run_app.py
```

## Сборка `.exe`

```bat
build_windows.bat
```

Готовое приложение будет здесь:

```text
dist\OfflineDialogueLabelerPro\OfflineDialogueLabelerPro.exe
```

Разметчику нужно передавать **всю папку** `dist\OfflineDialogueLabelerPro\`, а не только `.exe`.

## Если корпоративный PyPI не отдает пакеты

Создай рядом с проектом папку `wheelhouse/` и положи туда нужные wheel-файлы.
Тогда `build_windows.bat` попробует установить зависимости из неё.

Минимально нужны:
- `openpyxl==3.0.10`
- `pyinstaller==5.13.2`
- и транзитивные зависимости PyInstaller для Windows

## Примечание про Python у разметчика

**Python на машине разметчика не нужен.**
Он нужен только на машине, где ты **собираешь** `.exe`.

## Демо-режимы

В `config/modes.json` положен полный демо-конфиг с несколькими режимами и правилами.
В `input/` положен пример Excel-файла, на котором можно проверить интерфейс.
